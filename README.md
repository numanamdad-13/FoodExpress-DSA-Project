# FoodExpress Dispatch System

A console-based Food Delivery Dispatch System developed in C++ as a Data Structures and Algorithms (DSA) Lab Project.

## Features

- Order Scheduling using Max Heap
- Rider Assignment using Min Heap
- Kitchen Management using Queue
- Fast Order Search using Hash Map
- City Routing using Graph and Dijkstra's Algorithm
- Order Tracking
- File Handling
- Admin, Rider, and Customer Portals

## Data Structures Used

- Max Heap
- Min Heap
- Queue
- Hash Map
- Graph
- Dynamic Array
- Stack (Undo Feature)

## Technologies

- C++
- Object-Oriented Programming
- Visual Studio 2022

## Project Structure

- `main.cpp`
- `foodexpress.cpp`
- `FoodExpress.h`

## How to Run

1. Open the project in Visual Studio 2022.
2. Build the project.
3. Run the application.
4. Choose Administrator, Rider, or Customer.
